About this site

This site was designed by AI, inspired by my resume, and is self-hosted on an old MacBook Pro (circa 2011) running nginx in a docker container.

I was inspired to self-host when taking linux networking classes, and achieved this through a static IP reservation on the router with NAT forwarding.

Lovable originally wanted to make this a react site, but a static html can be served 20x faster, and who in the world needs a portfolio site to be dynamically generated on the fly.
